# Change Log

## 1.1.0

`2022-05-19`

- 增加 `showPracticalBranchNode` 和 `showPracticalBranchRemove` 属性，支持显示实际的分支节点

## 1.2.0

`2022-05-24`

- 节点的默认 id 从 `node-` 开头修改为 `节点类型-` 开头

## 1.3.0

`2022-06-09`

- 增加 `registerRemoteNodes` 属性，支持远程节点注册，通过 System.js 加载对应的 js / css 资源

## 1.4.0

`2022-08-04`

- 增加 `sortable` 属性，支持条件节点之间进行拖拽排序

## 1.4.1

`2022-08-04`

- 优化拖拽排序过程中的虚拟边框
